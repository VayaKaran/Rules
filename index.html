1-Minute Quiz Rules

  • ==> You have 60 seconds to answer as many questions as possible.
  • ==> Each correct answer earns 10 points.
  • ==> -3 points for each wrong answer
  • ==> Use the 'Next' button to move to the next question.
  • ==> Time will stop automatically when the quiz ends.
  • ==> No external help is allowed
  • ==> If time runs out while responding answer than no points will be added
